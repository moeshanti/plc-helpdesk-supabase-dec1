A modern, AI-powered HelpDesk system designed for 1C ERP support.

## Features
- **Smart Assistant**: Video/Image analysis to auto-fill tickets.
- **Real-time Notifications**: Instant alerts for assignments and updates.
- **SLA Tracking**: Visual health cards and compliance gauges.
- **Supabase Integration**: Secure Auth, Database, and Storage.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` and Supabase credentials in [.env.local](.env.local)
3. Run the app:
   `npm run dev`
